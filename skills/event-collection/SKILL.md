# Event Collection

Use this skill when the user provides or asks to plan travel details. It returns structured trip fields for downstream planning.

## Output Contract

- `origin`
- `destination`
- `start_date`
- `end_date`
- `trip_purpose`
- `missing_info`
- `extracted_count`
